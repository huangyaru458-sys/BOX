# 简单的设计文档

## 1. 项目概述
本项目采用 `ESP32 + MQTT + Node-RED + SQLite + Web` 架构，实现旧衣物回收箱的状态采集、远程控制、告警联动、离线应急和历史数据查询。

## 2. 功能目标
1. 实时采集温湿度、空气质量、容量、人体接近、姿态倾斜等状态。
2. 支持前门、后门、风扇、蜂鸣器、告警等控制。
3. 支持 Web 端、Node-RED Dashboard、串口、MQTT 四种链路查看和控制。
4. 支持断线重连和离线模式。

## 6. 控制与告警策略
1. 取消告警后进入 5 分钟抑制窗口。
2. 离线状态禁止远程控制。
3. 前门、后门、风扇、告警状态需由设备回传后同步前端。
4. 传感器数据以设备上报为准，前端、串口、OLED、Node-RED 调试窗口保持一致。

## 7. 验收点
1. 前门可开关。
2. 后门可开关。
3. 风扇可开关。
4. 蜂鸣器可控制。
5. 键盘可本地开锁。
6. 告警可取消并抑制 5 分钟。
7. MQTT 可连接并自动重连。
8. 超声波按 `GPIO2 / GPIO35` 正常采集。
9. 传感器数据显示与后端、串口、OLED 一致。
